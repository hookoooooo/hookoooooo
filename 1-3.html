蘿蔔糕

材料：

  • 白蘿蔔
  • 米粉
  • 蝦米
  • 香菇
  • 醬油
  • 糖
  • 鹽
    • <
  • 清水
  • 油
<

步驟：

  1. 將白蘿蔔削皮，切成絲狀或小丁，備用。
  2. 將米粉用清水泡發，備用。
    3. <
  3. 將蝦米和香菇剁碎，備用。
    4. <
  4. 在鍋中燒熱油，加入蝦米和香菇炒香。
    5. <
  5. 加入白蘿蔔繼續炒熟，調味鹽、糖、醬油。
    6. <
  6. 將泡發好的米粉加入鍋中，不斷攪拌均勻。
    7. <
  7. 將混合物倒入蒸模中，放入蒸鍋中蒸熟，直到蘿蔔糕凝固。
  8. 待蘿蔔糕冷卻後，切成所需形狀，可切片或切丁。
  9. 可選擇將蘿蔔糕片煎至表面金黃，增添香脆口感。

典故

無論過年拜拜或是送禮，蘿蔔糕都是年菜及年禮的好選擇，
因為「糕」與「高」同音，而有「步步高昇」、「年高長壽」等祝賀意涵；
因舊時禾穀一年成熟一次，亦有五穀豐收的意思。 蘿蔔糕就是菜頭粿，菜頭的台語發音有「好彩頭」

寓意，菜頭也是年前豐富盛產的蔬菜，符合經濟效益。
